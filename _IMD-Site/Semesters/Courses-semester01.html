Digital Citizenship

In this class, students will focus on the social and the socio-cultural aspects of learning. As well as studying and time management skills. Networking, community building, and Group work. This course has three delivery modes : In class face-to-face, half in class half online, and fully online.

Introduction to DSLR

Students will learn how to use DSLR cameras in this class.Through a hands on approach, students will learn the theory of photography as well as the techniques and the technology. Students will learn how to avoid post-production problems by using best practices during a shoot.

Responsive Web Design

During this class students will be introduced to responsive web design using the HTML and CSS coding languages to build sites that work on any device from an iphone 5 to a 20'' computer monitor. Students will use best practices in their coding techniques and web design.

Digital Graphics

Students will gain an understanding of the modern art techniques, file formats and tools used in the graphic design industry. Students will become accustomed with planning and executing digital graphics projects. The best practices will be taught for file format and size and how to properly export assets for online use and print.

Design Foundations

In this class, students will get a hands-on approach to design theory. Students will be introduced to the fundamental design rules, layout principles colour theory, typography, UIs and interactive design.
